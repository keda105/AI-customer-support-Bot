# 💅 AI Beauty Salon Assistant (Voiceflow Chatbot)

This is an intelligent AI-powered chatbot designed to automate customer interactions for beauty salons, hair studios, and spas. Built with **Voiceflow**, this bot streamlines the booking process and provides instant customer support.

## ✨ Key Features
- **📅 Automated Bookings:** Helps clients select services and book appointments 24/7.
- **💄 Service Menu:** Displays detailed descriptions of treatments (Hair, Nails, Makeup, etc.).
- **🕒 Info & Hours:** Provides instant access to salon location, contact details, and working hours.
- **❓ Smart FAQ:** Answers common questions about pricing, cancellation policies, and prep-care.
- **📱 Mobile Optimized:** A seamless and responsive interface for both mobile and desktop users.

---

## 🛠️ Tech Stack
- **Voiceflow:** Conversation design and UX logic.
- **AI Logic:** Conditional flows and natural language processing to guide the user.

## 📂 How to Install & Use
If you want to use or modify this project:
1. Download the `.vf` file from this repository.
2. Log in to your [Voiceflow](https://www.voiceflow.com) account.
3. Click **"Import"** and select the `.vf` file.
4. Hit **"Publish"** and embed the widget into your website (Wix, WordPress, Shopify, etc.).
